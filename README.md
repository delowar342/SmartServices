# SmartServices — Home & FM Platform

> Bangladesh's first unified home maintenance and facilities management platform.

## Live Demo

👉 [View Prototype](https://yourusername.github.io/smartservices)

## What's Inside

This is a fully interactive HTML prototype with 5 portals:

| Portal | Description |
|--------|-------------|
| 🏠 Landing Page | Public website with service catalog, pricing, and CTAs |
| 👤 Customer Portal | Dashboard, booking, job tracking, invoices, payments, feedback |
| ⚙️ Admin Dashboard | Dispatch, accounts, billing, collections, analytics, HR, inventory |
| 🔧 Technician App | Job queue, parts request, history, earnings |
| 💵 Collector App | Cash collection schedule and confirmation |

## How to Run Locally

Just open `index.html` in any browser — no build process, no dependencies.

```bash
open index.html
```

## How to Deploy on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://yourusername.github.io/smartservices`

## Tech Stack

- Pure HTML, CSS, JavaScript — zero dependencies
- Google Fonts (Playfair Display, Outfit, JetBrains Mono)
- Self-contained single file — works offline
